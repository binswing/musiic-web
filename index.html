Commands

These are some useful commands, and the default command prefix (cp) is !

Command prefix(cp)

@musiic.ml

Recent command prefix

(cp)changeprefix [new cp]

Change the command prefix for your server

Connection

(cp)join/(cp)connect

Connect bot to the voice channel

(cp)leave/(cp)disconnect

Disconnect bot from the voice channel

Play

(cp)play [url]

Play audio in youtube with url

(cp)play [keywords]

Play audio in youtube with keywords

Search

(cp)search [keywords]

Search top 10 audios in youtube with keywords

Help

(cp)help

Show commands list

Others

(cp)pause/(cp)stop

Stop audio

(cp)resume/(cp)continued

Resume audio
